TabScore ReadMe file

TabScore is designed for use on a PC with Windows 10 (which includes Internet
Information Services (IIS) 10),  .NET Framework 4.6.1 and ASP.NET 4.7

Please ensure you have completed all the steps in the TabScore installation
guide before attempting to run TabScore

TabScore is limited to 4 section (A, B, C and D) and 30 tables per section
